# Jude Wade - Sound Designer Portfolio

A modern, interactive portfolio website showcasing the work of Jude Wade, a professional sound designer and audio engineer.

## Features

- **Interactive Launch Animation** - Letter-by-letter reveal of the JUDE logo
- **Dynamic Audio Waves** - Real-time sine wave visualization with mouse interaction
- **Responsive Design** - Optimized for all screen sizes
- **Resume & Certifications** - Professional credentials display
- **Smooth Animations** - Modern UI transitions and effects

## Tech Stack

- **Next.js 15** - React framework with TypeScript
- **Tailwind CSS 4** - Utility-first CSS framework
- **Custom CSS** - Semantic class structure with CSS variables
- **SVG Animations** - Interactive graphics and visualizations

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/jdoseph/judewade-portfolio.git
cd judewade-portfolio
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Development

```bash
npm run dev          # Start development server
npm run build        # Build for production
npm run start        # Start production server
npm run lint         # Run ESLint
```
